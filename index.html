descripción

Sistema de automatización del hogar con agente IA conversacional en español. Procesa comandos de voz mediante Gemini API y ejecuta acciones sobre dispositivos IoT reales en tiempo real vía MQTT.

desafío principal

Integrar tres capas radicalmente distintas — LLM en la nube, broker MQTT y Home Assistant — coordinando autenticación, latencia y errores. Todo desde Google Cloud Shell sin PC local.

solución

Arquitectura de 3 microservicios: (1) API Gateway FastAPI para recibir comandos transcritos, (2) Agente LLM con Gemini que interpreta intención y genera instrucciones estructuradas, (3) Servicio MQTT que publica al dispositivo en Home Assistant. Docker Compose local + Kubernetes en producción + CI/CD con GitHub Actions.

aprendizajes clave

  • Diseño de agentes IA en producción real
  • Orquestación Kubernetes sin recursos locales
  • Prompt engineering para acciones hardware
  • Gestión de secretos y configuración segura en GKE
  • Documentación técnica + plan de negocio

herramientas

  • Python 3.12 · FastAPI · Pydantic
  • Gemini API · LangChain · CrewAI
  • MQTT · Mosquitto · Home Assistant
  • Docker · Kubernetes · GKE
  • GitHub Actions · Cloud Run

habilidades aplicadas

Arquitectura de software Python avanzado APIs REST Google Cloud Platform Containerización Kubernetes Protocolos IoT Prompt Engineering Agentes LLM DevOps / CI-CD Documentación técnica Pensamiento sistémico
